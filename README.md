# internBuddy 

## Description
internBuddy is an application where users are matched to internships based on their skills and the skills required by the internship posting. 
With intern buddy, users are able to:
 * ⭐️ Create an account, customized with their skills, major, and university.
 * ⭐️ Get matched with internships that match their skills.
 * ⭐️ Learn about possible alumni that went to the same university and who work at the specified internship posting they're looking at.
 * ⭐️ Add more skills to generate more possible internship matches.
 * ⭐️ View profile information.
 * ⭐️ Update profile information.
 * ⭐️ Delete their account if needed.

## Running internBuddy
Use the terminal to run the application:
1. Clone the repository
2. To run the backend:
   - run "cd backend"
   - run "python3 main.py"
4. To run the frontend:
   - run "cd frontend"
   - run "npm run dev"
5. Run the application using local host 5173, the link should be provided in the terminal once you run "npm run dev" in step 4.
